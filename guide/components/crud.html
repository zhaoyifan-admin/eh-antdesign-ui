多功能表格 Crud ​

何时使用 ​

  • 当有大量结构化的数据需要展现时；
  • 当需要对数据进行排序、搜索、分页、自定义操作等复杂行为时。

如何使用 ​

vue

<script setup lang="ts">
  const columns = [
    {
      title: 'Name',
      dataIndex: 'name',
      sorter: true,
    },
    ...
  ]
</script>
<template>
  <eh-Crud :columns="columns" :options="..."></eh-Crud>
</template>

基本使用 ​

Show Code
vue

<script setup lang="ts">
  const columns = [
    {
      title: 'Name',
      dataIndex: 'name',
    },
    {
      title: 'Gender',
      dataIndex: 'gender',
    },
    {
      title: 'Email',
      dataIndex: 'email',
    },
  ]
</script>
<template>
  <eh-Crud :columns="columns" :options="{
      bordered: true
  }"></eh-Crud>
</template>
