*用于反馈一系列操作任务的处理结果*

## 何时使用

- 当有重要操作需告知用户处理结果，且反馈内容较为复杂时使用

## APIs

参数 | 说明 | 类型 | 默认值 | 必传
-- | -- | -- | -- | --
status | 结果的状态，决定图标和颜色 | 'success' &#124; 'error' &#124; 'info' &#124; 'warning' &#124; '404' &#124; '403' &#124; '500' | 'info' | false
title | 标题文字 | string &#124; slot | '' | false
subTitle | 副标题文字 | string &#124; slot | '' | false
